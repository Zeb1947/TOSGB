Step 1




Download Neon Drift here or purchase a cd installation from Hikari (The developer of this game and this website)


Step 2




Run the game installer, it should appear as an .exe installer.

If you downloaded the linux/mac/python build, install WinRar and extract the .rar file, then skip straight to Step 5


Step 3




Accept the License Agreement

Then click next.





An Informative Dialog will then open, read and click next the continue.





A dialog will open straight after the last, it will ask for a install path, the default is fine. As shown in the image above it requires at least 568.2MB of disk space to continue. After checking/completing these things/steps click Next.




A dialog will open asking if you would like a desktop shortcut. This is optional but recomended. After considering if you would like a desktop shortcut or not click next.




This dialog is just a conformation for install. From here just click install




You should now have a screen like this, please wait patiently while Neon Drift installs.




An Informative Dialog will open after this. Just click next.




From here it is very simple. Click finish.




An icon like this should appear on your desktop. Double click it to run it. It may take a bit to open but rest assured it will run.




If you did not select a desktop shortcut, search Neon Drift in the search bar like Step 1 Step 2 in This Guide
